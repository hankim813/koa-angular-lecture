# Boot It Up

Clone the [koa-boot](https://github.com/hankim813/koa-boot) repo, remove the git directory, and re-init!:

```
$ git clone git@github.com:hankim813/koa-boot.git
$ cd koa-boot/
$ rm -rf .git
$ git init
```

## Usage

```
$ make
```
